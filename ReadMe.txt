"Offset Gaussian difference" method approach based on fabric defect detection will be used.
C++ software language will be used in the system targeting 11 different error types.
At the end of the project, online image analysis is performed. By accessing the /image folder, offline image analysis results are given.
This folder contains detected images, Doog filter results, threshold histogram results, dill result, erod result and Normalized image.
Detection, DoG filter, feature extraction, histogram, imerode are coded separately.
Libraries it has: Opencv.
Time module has been created for daily data generation.
Upload the image you want to be detected to the "/images" folder.(RGB)
